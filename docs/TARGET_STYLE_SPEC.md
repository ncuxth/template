# 蓝色紧凑中文技术简历模板规范

## 页面尺寸与边距

- 页面保持 A4 画布：`210mm x 297mm`。
- 继续使用现有页边距 CSS 变量（`--page-margin-x`、`--page-margin-y`）。
- 渲染后继续通过 `fitToOnePage()` 做单页缩放适配。

## 头部区域

- `# 姓名` 渲染为头部姓名，居中显示。
- 从 `# 姓名` 到第一个 `##` 之间的普通文本行渲染为多行个人信息（每行一个 `.resume-meta-line`）。
- 保持头部信息紧凑，优先保证单页容纳。

## 头像规则

- 头部区域内 Markdown 图片语法（例如 `![avatar](...)`）解析为右上角头像。
- 头像元素使用 `.resume-avatar`，固定尺寸、边框、`object-fit: cover`。
- 若未提供头像语法，则不渲染头像节点。

## 分区标题规则

- 二级标题 `##` 渲染为分区标题。
- 标题文本使用蓝色（`.section-title` 在蓝色模板下覆盖）。
- 标题下方横线为蓝色，保持紧凑分区间距。

## 条目行规则

- 行尾日期识别后渲染为 `.entry-row`：左侧 `.entry-title`，右侧 `.entry-date`。
- 支持日期格式：
  - `（2026-01 - 2026-04）`
  - `(2026-01 - 2026-04)`
  - `2026-01 - 2026-04`
  - 兼容原有 `yyyy.mm` 与 `至今/现在/Present`。

## 项目结构规则

- 项目经历分区内按项目块渲染：每个项目容器为 `.project-block`。
- `---` 作为项目边界，触发当前项目块闭合，并进入下一个项目块。
- 项目块之间可通过样式增加分隔线和间距。

## 列表规则

- 继续支持有序/无序列表。
- 支持二级 bullet（通过前导缩进与 `-`/`*`）。
- 蓝色紧凑模板下列表行距与段距更紧凑。

## 打印规则

- 打印模式保持 A4 输出，控制面板隐藏。
- 保持日期右对齐与内容可读性。
- 不改动现有 `@page` 规则与打印结构。
