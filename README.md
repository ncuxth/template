# Markdown 简历模板使用说明

这是一个本地 Markdown 简历渲染器，可以把 `.md` 简历内容实时渲染成 A4 简历模板，并支持调节页边距、字号、字体、行距、段落间距和普通文字颜色深浅。

## 使用方法

1. 双击打开 `index.html`。
2. 点击右侧 `选择MD` 按钮。
3. 选择你的 Markdown 简历文件，例如 `resume.md`。
4. 页面会自动把 Markdown 渲染成左侧 A4 简历预览。
5. 在右侧调节页边距、字体、字号、行距、段落间距等参数。
6. 点击右上角 `打印`，在浏览器打印窗口中选择“另存为 PDF”即可导出。

## Markdown 写法

一级标题会作为姓名：

```md
# 郭家硕
```

姓名下面的第一段会作为联系方式：

```md
**Phone**:15305145683｜**Email**:2318891066@qq.com | **Blog**:daoge.online
```

二级标题会作为简历分区：

```md
## 教育经历
## 项目经验
## 技能与证书
```

行尾日期会自动靠右：

```md
**南京邮电大学 - 本科  `双一流`    （2024.09-2028.07）**
```

反引号内容会渲染成蓝色标签：

```md
`双一流`
```

项目之间可以用分割线：

```md
---
```

加粗内容使用 Markdown 标准写法：

```md
这是**重点内容**
```

## 注意事项

- `选择MD` 只负责读取文件到页面中，不会自动写回原 Markdown 文件。
- 如果在网页中修改了 Markdown 内容，需要自行复制回原 `.md` 文件。
- 当前模板会尽量把内容缩放到一张 A4 纸内，字号或行距过大时内容会自动缩小。
